# LSCO - コーポレートサイト

## プロジェクト概要

LSCOは、企業向けのモダンでレスポンシブなコーポレートウェブサイトです。お問い合わせフォームを備え、PC、タブレット、スマートフォンなど様々なデバイスに最適化されたデザインを実現しています。

## 主な機能

### ✅ 実装済み機能

1. **レスポンシブデザイン**
   - PC（1200px以上）
   - タブレット（768px-1024px）
   - スマートフォン（768px以下）
   - 横向きスマートフォン対応

2. **モダンなUI/UX**
   - ヘッダー固定表示
   - スムーズスクロール
   - モバイルメニュー
   - スクロール時のヘッダー制御

3. **お問い合わせフォーム**
   - 必須項目バリデーション
   - メールアドレス形式チェック
   - 電話番号形式チェック
   - リアルタイムエラー表示
   - 送信完了メッセージ

4. **セクション構成**
   - ヒーローセクション
   - 会社概要
   - サービス紹介
   - お問い合わせ
   - フッター

5. **アクセシビリティ**
   - セマンティックHTML
   - ARIAラベル
   - キーボードナビゲーション対応
   - スクリーンリーダー対応

## 使用方法

### 基本的な使い方
1. `index.html`をブラウザで開く
2. ナビゲーションメニューから各セクションに移動
3. お問い合わせフォームから問い合わせ送信

### カスタマイズ方法

#### 色の変更
`css/style.css`のCSS変数を変更:
```css
/* プライマリカラー */
color: #2563eb;

/* グラデーション背景 */
background: linear-gradient(135deg, #667eea 0%, #764ba2 100%);
```

#### フォントの変更
`index.html`のheadセクションでGoogle Fontsを変更:
```html
<link href="https://fonts.googleapis.com/css2?family=Noto+Sans+JP:wght@300;400;500;700&display=swap" rel="stylesheet">
```

#### コンテンツの更新
- 会社情報: `index.html`内の該当セクションを編集
- 連絡先: フッターとお問い合わせセクションの情報を更新
- サービス内容: `.service-card`内のコンテンツを変更

## トラブルシューティング

### モバイルメニューが動作しない
- JavaScriptが有効になっているか確認
- ブラウザのコンソールでエラーを確認

### フォームが送信できない
- 必須項目が全て入力されているか確認
- メールアドレスの形式が正しいか確認
- ブラウザのバリデーションメッセージを確認

### レイアウトが崩れる
- ビューポートメタタグが設定されているか確認
- CSSファイルが正しく読み込まれているか確認
